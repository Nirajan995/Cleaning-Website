# Getting started
- Clone the repository
```
git clone  <github template url> <project_name>
```
- Install dependencies backend
```
cd backend
npm install
```
- Install dependencies backend
```
cd frontend
npm install
```
- Build and run the project
```
npm start
```
  Navigate to `http://localhost:3000`
